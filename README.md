# Autonomous-Vehicle

This repository contains code for an autonomous vehicle. With this project, I wired together the motors and the sensors on a breadboard, all with the Arduino Uno

The vehicle has the following functions: Bluetooth driving, line tracking, and object avoidance.

Using the ArduinoBlue App, you can operate the vehicle through Bluetooth and enable/disable line tracking and object avoidance.

Video Demonstration: https://youtu.be/eqzsAxDyG-o
